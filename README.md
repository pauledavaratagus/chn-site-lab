# chn-site-lab

## 项目简介

chn-site-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，主要用于存放各类静态 HTML 文件，便于版本管理与分发。

## 目录说明

```
chn-site-lab/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件代表一个独立的 HTML 页面，命名清晰以便识别。
- **assets/**：各页面共享或专用的资源文件，按需组织。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中运行。
- 页面可能引用 `assets/` 中的资源，请保持相对路径正确。
- 归档内容可能包含示例、演示、工具等，具体用途见各页面内部说明。

## 维护说明

- 欢迎提交 Pull Request 添加或更新页面。
- 新增页面请放置在 `pages/` 目录下，并确保资源引用正确。
- 不建议在页面中加入外部跟踪、广告或商业内容。
- 如有问题，请通过 Issues 反馈。

## 许可

本仓库内容按需使用，无特定许可证约束，但请尊重原始作者权利。
